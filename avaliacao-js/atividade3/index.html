Exemplo document.getElementById()

É usada para obter uma referência a um elemento específico do DOM com base no valor do atributo 'id' desse elemento.

Exemplo document.getElementsByTagName()

Este método retorna uma lista de todos os elementos com o nome da tag especificada. Como no exemplo, se você quiser obter todos os elementos 'h1' em um documento, você o usa.

Exemplo document.getElementsByClassName()

Este método retorna uma lista de todos os elementos que possuem a classe CSS especificada, como no exemplo.

Exemplo document.querySelector()

Este método retorna a primeira ocorrência de um elemento que corresponde a um seletor CSS especificado. Ele retorna apenas um elemento, diferente dos outros que retornam a mais de um.

Exemplo document.querySelectorAll()

Este método retorna todos os elementos que correspondem a um seletor CSS especificado, em forma de NodeList.
